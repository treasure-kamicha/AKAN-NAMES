## AKAN-NAMES

Initial dev - Treasure kamicha

## DESCRIPTION

The Akan Name Generator is a simple web application to know a kid's name since  a child’s name is often determined by the day of the week they were born. 
This project calculates the day of the week from the user’s birth date and displays the corresponding Akan name.

## TECH STACKS 

HTML
CSS
JAVASCRIPT

## USAGE

Launch the web page in your browser.
Use the date picker or type in your date of birth.
Choose either Male or Female from the options provided.
Click the Generate Name button.
The  app will calculate the day of the week you were born.
It will display your Akan name based on that day and gender.
